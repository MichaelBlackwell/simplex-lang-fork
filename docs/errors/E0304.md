error[E0304]: yield outside of generator

This error occurs when you use the `yield` keyword outside of a generator
function.

Example of erroneous code:
    fn main() {
        yield 42;  // error: yield outside of generator
    }

    fn process(x: i64) {
        if x > 0 {
            yield x;  // error: yield outside of generator
        }
    }

Common causes:
- Using yield in a regular function instead of a generator
- Forgetting to mark a function as a generator
- Misunderstanding generator syntax

To fix this, define a generator function:

Correct examples:
    gen fn count_up(n: i64) -> i64 {
        let i = 0;
        while i < n {
            yield i;  // ok: inside generator function
            i = i + 1;
        }
    }

    fn main() {
        for value in count_up(5) {
            println(value);  // prints 0, 1, 2, 3, 4
        }
    }

    gen fn fibonacci() -> i64 {
        let a = 0;
        let b = 1;
        loop {
            yield a;  // ok: inside generator
            let next = a + b;
            a = b;
            b = next;
        }
    }

Understanding generators:
- `gen fn` defines a generator function
- `yield` produces a value and suspends the generator
- Generators implement Iterator and can be used in for loops
- Each call to next() resumes execution until the next yield

If you want to return multiple values without a generator,
consider returning an array or using a callback:
    fn get_values() -> [i64] {
        [1, 2, 3, 4, 5]  // return array instead
    }
