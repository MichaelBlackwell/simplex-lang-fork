error[E0300]: break outside of loop

This error occurs when you use the `break` keyword outside of a loop
construct (while, for, loop).

Example of erroneous code:
    fn main() {
        break;  // error: `break` outside of loop
    }

    fn process(x: i64) {
        if x > 10 {
            break;  // error: `break` outside of loop (if is not a loop)
        }
    }

    fn example() {
        let f = || {
            break;  // error: cannot break from inside a closure
        };
    }

Common causes:
- Using break at the top level of a function
- Confusing if statements with loops
- Trying to break from within a closure to an outer loop
- Accidentally deleting the surrounding loop

To fix this, use break only inside loops:

Correct examples:
    fn main() {
        while true {
            if should_stop() {
                break;  // ok: inside while loop
            }
        }
    }

    fn process(items: [i64]) {
        for item in items {
            if item == 0 {
                break;  // ok: inside for loop
            }
        }
    }

    fn search() {
        loop {
            if found() {
                break;  // ok: inside loop
            }
        }
    }

If you want to exit a function early, use return:
    fn process(x: i64) -> i64 {
        if x > 10 {
            return 0;  // ok: early return from function
        }
        x * 2
    }

See also:
- E0301: continue outside of loop
- E0302: return in closure
