error[E0009]: invalid attribute

This error occurs when an attribute (annotation starting with `@`) is
malformed or not recognized.

Example of erroneous code:
    @unknown_attr           // error: unrecognized attribute
    fn main() {
    }

    @inline()               // error: @inline doesn't take arguments
    fn compute() {
    }

    @                       // error: expected attribute name
    fn other() {
    }

Common causes:
- Using an attribute that doesn't exist
- Wrong syntax for attribute arguments
- Attribute placed in invalid position
- Typo in attribute name

To fix this, use a valid attribute with correct syntax:

Valid attributes in Simplex:
- @inline - suggest inlining the function
- @noinline - prevent inlining
- @test - mark as a test function
- @bench - mark as a benchmark function
- @deprecated - mark as deprecated
- @export - export symbol for FFI

Correct examples:
    @inline
    fn fast_function() -> i64 {
        42
    }

    @test
    fn test_addition() {
        assert(1 + 1 == 2);
    }

    @deprecated
    fn old_api() {
        // ...
    }

See also:
- E0001: unexpected token
