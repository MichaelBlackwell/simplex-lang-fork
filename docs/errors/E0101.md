error[E0101]: cannot find function in this scope

This error occurs when you try to call a function that hasn't been defined
or imported into the current scope.

Example of erroneous code:
    fn main() {
        let result = calculate(5);  // error: cannot find function `calculate`
    }

    fn helper() {
        compute();  // error: `compute` is not defined
    }

Common causes:
- Calling a function that doesn't exist
- Typo in function name
- Forgetting to import the module containing the function
- Function defined in a different module without proper import

To fix this:
1. Define the function before calling it
2. Import the module containing the function
3. Check for typos in the function name

Correct examples:
    fn calculate(x: i64) -> i64 {
        x * 2
    }

    fn main() {
        let result = calculate(5);  // ok: function is defined
    }

For module imports:
    use math::sqrt;  // import sqrt from math module

    fn main() {
        let x = sqrt(16.0);  // ok: function is imported
    }

If you meant to use a method, use dot notation:
    struct Point { x: i64, y: i64 }

    impl Point {
        fn distance(self) -> f64 { /* ... */ }
    }

    fn main() {
        let p = Point { x: 3, y: 4 };
        let d = p.distance();  // ok: method call
    }

See also:
- E0100: cannot find value in this scope
- E0107: no method found
