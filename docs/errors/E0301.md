error[E0301]: continue outside of loop

This error occurs when you use the `continue` keyword outside of a loop
construct (while, for, loop).

Example of erroneous code:
    fn main() {
        continue;  // error: `continue` outside of loop
    }

    fn process(x: i64) {
        if x > 10 {
            continue;  // error: `continue` outside of loop
        }
    }

    fn example() {
        let f = || {
            continue;  // error: cannot continue from inside a closure
        };
    }

Common causes:
- Using continue at the top level of a function
- Confusing if statements with loops
- Trying to continue from within a closure to an outer loop
- Accidentally deleting the surrounding loop

To fix this, use continue only inside loops:

Correct examples:
    fn main() {
        let i = 0;
        while i < 10 {
            i = i + 1;
            if i == 5 {
                continue;  // ok: skip to next iteration
            }
            println(i);
        }
    }

    fn process(items: [i64]) {
        for item in items {
            if item < 0 {
                continue;  // ok: skip negative numbers
            }
            process_positive(item);
        }
    }

If you want to skip the rest of a function, use return:
    fn process(x: i64) {
        if x > 10 {
            return;  // ok: early return
        }
        // process x...
    }

See also:
- E0300: break outside of loop
- E0302: return in closure
