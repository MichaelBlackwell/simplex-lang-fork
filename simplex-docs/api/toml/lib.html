fn toml_parse

fn toml_parse(content: String) -> i64 {

simplex-toml - TOML parsing and serialization for Simplex Parse and generate TOML configuration files. # Example ```simplex use modulus::simplex_toml::toml fn main() -> i64 { let content: String = "[package]\nname = \"my-app\"\nversion = \"1.0.0\"" let config: i64 = toml_parse(content) if config != 0 { let name: String = toml_get_string(config, "package.name") println(name) // "my-app" toml_free(config) } 0 } ``` Parse a TOML string into a table Returns 0 on parse error, otherwise a table handle
