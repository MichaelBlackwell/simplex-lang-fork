fn block_on

fn block_on<F, T>(future: F) -> T

Run an async function to completion on the current thread. This is the main entry point for async programs in Simplex. It blocks the current thread until the future completes. # Example ```simplex runtime::block_on(async_main()); ```

fn spawn

fn spawn<F>(future: F) -> JoinHandle<F::Output>

Spawn an async task to run concurrently. The task will run on the runtime's thread pool. # Example ```simplex spawn(async { do_background_work().await; }); ```
